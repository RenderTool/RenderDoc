导读

创建复杂的用户界面时，可能需要重用大量模板化的小部件，例如：切换菜单时能不能通过一个模板实现修改部分内容，而保持部分内容不变。
命名槽小组件-NameSlot专门用于在模板化 UI 中充当占位符。
