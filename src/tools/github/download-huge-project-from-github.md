# Github- 仓库文件下载慢

## 问题详情

在 Github 上阅读看[PBRTV3](“https://github.com/mmp/pbrt-v3”)不是很方便，所以就想下载下来阅读。
结果下载时下载速度总是只有 4kb/s - 6kb/s，下载到最后还会报错……

## 解决方案

1. 梯子，国内比较敏感这个就自己解决吧。
2. 使用 Gitee（码云）的时候提供了下载 Github 项目的服务。
